# 隐私声明

## 本地文件

MD+HTML Reader 仅对你明确选择的文件夹进行操作。编辑、评论、文件搜索、内容搜索和 HTML 导出都在本地运行。这些功能不需要账号。

评论以与文档路径关联的侧车数据形式存储，旨在随你选择的工作区保留。

## 可选 AI 服务

AI 功能完全可选。在你选择服务并批准单次请求前，不会使用 AI 服务。

生成 AI 阅读版时，应用会将当前 Markdown 文档发送给你选择的服务商。生成文档建议或优化稿时，应用会发送当前 Markdown 及该文档尚未解决的评论。应用不会自动发送整个工作区。

你有责任选择服务商，并确保自己有权向其发送文档内容。使用前请查看该服务商的条款与隐私政策。

## API 凭据

对于兼容 OpenAI 的服务，基础 URL 和所选模型可保存在本机以方便使用。API 密钥仅保存在当前应用会话的内存中，本应用不会将其写入磁盘。

## 数据共享与支持

应用没有内置分析系统或账号系统。若你通过外部渠道联系项目团队，该渠道的隐私条款将适用。若数据处理方式发生变化，本声明会在公开生产版本发布前更新。
